# Мирас — Калькулятор наследования РК

Современный сайт для расчёта наследования по закону Республики Казахстан.

## 🚀 Запуск

### Локально
Откройте `index.html` в браузере — всё работает без сервера.

### Netlify (5 минут)
1. Зайдите на [netlify.com](https://netlify.com)
2. Перетащите папку `miras` в область деплоя
3. Готово!

### GitHub Pages
1. Создайте репозиторий на GitHub
2. Загрузите все файлы
3. Settings → Pages → ветка `main` → корень `/`

## 📁 Структура

```
miras/
├── index.html          ← Главная страница
├── favicon.svg         ← Иконка
├── css/
│   └── style.css       ← Все стили
└── js/
    ├── i18n.js         ← Переводы RU / KK
    ├── auth.js         ← Авторизация (localStorage)
    └── app.js          ← Логика калькулятора
```

## ✨ Функционал

- **Главная** — описание сервиса, 3 ключевых принципа
- **Закон РК** — очереди наследников, статьи ГК РК, особенности
- **Калькулятор** — дерево наследования с поиском по ФИО
- **Авторизация** — регистрация, вход, профиль с аватаркой
- **Язык** — переключение РУ / ҚАЗ (казахский)

## 🔐 Безопасность

Авторизация хранит данные в `localStorage` (демо-режим).  
Для продакшена подключите Firebase Auth, Supabase или Auth0.

## ⚖ Данные

Основано на Гражданском кодексе РК (статьи 1040–1083).  
Информация носит справочный характер.
